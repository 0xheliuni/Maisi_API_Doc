## 关于渠道来源 

模型供应商一般分 4 种类型的渠道：**官方**、**合作商家**、**逆向**、**默认**。 不同的供应商渠道价格也不同，官方 > 合作商家 > 逆向，逆向渠道的价格最低，但是稳定性和功能性也最差。

市面上很多中转站为了成本考虑，会混用供应商渠道，导致接口不稳定，价格不透明，甚至有安全隐患。

我们明码标价，您可以随时切换您想要使用的渠道，享受不同的折扣。

### **官方** 

顾名思义，就是模型供应商自己提供的服务。

### **合作商家** 

每家模型供应商除了自己**官方**提供的服务外，还和其他厂商合作，例如：**OpenAI**和**Azure**合作，**Claude**和**亚马逊**以及**谷歌**合作。 但是这些合作商家在 API 功能和新模型上会有滞后性。

### **逆向** 

逆向渠道是指通过技术方法方式将模型官方客服端服务转换为 API 接口形式提供调用，这种 API 接口稳定性较差。
