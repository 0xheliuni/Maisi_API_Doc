# 渠道分组说明 

::: warning ⚠️ 注意
分组的区别主要是 OpenAI、Claude 模型对应的渠道来源不同，对应的价格也随之不同。

这就是为什么市面上有那么多价格不一致的原因，我们明码实价，绝对不搞虚的；令牌所使用的渠道分组，可随便更改。
:::

## 关于渠道 

### OpenAI 

渠道名称

说明

官方

纯官方 API，支持函数调用，稳定，支持高并发，无审核

默认

采用企业无审核 Azure OpenAI， 支持函数调用， 稳定，支持高并发，无审核

Azure

Azure OpenAI ，支持函数调用， 支持高并发， 有审核

逆向

ChatGPT 逆向接口， 不支持高并发，稳定性不保证

### Claude 

渠道名称

说明

稳定

纯官方 API，支持函数调用，稳定，支持高并发，无审核

默认

Google/AWS API， 支持函数调用， 稳定，支持高并发，无审核

逆向

Claude 逆向接口， 不支持高并发，稳定性不保证
