# Next Chat 

[GitHub](https://github.com/ChatGPTNextWeb/NextChat)

如果要兼容其它厂商模型，例如 claude、gemini 等，需要在 设置->自定义模型 中按如下格式添加自定义模型：+模型名称@OpenAI，

例如：+claude-3-opus-20240229@OpenAI

## 简单配置 

`https://{url}/#/?settings={"key":"{key}","url":"https://maisi-ai.com"}`

-   `{url}` 是您部署的服务器地址
-   `{key}` 是您的令牌

替换`{url}`和`{key}`后，在浏览器中打开即可使用
